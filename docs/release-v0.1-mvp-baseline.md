# v0.1 MVP Baseline Release Note

Date: 2026-05-10

This note marks MVP 1 through MVP 6 as the baseline that Alpha 0.1 stabilizes.
It is a release-note candidate rather than a git tag by itself.

## Baseline Scope

- MVP 1 Work Package backend foundation
- MVP 2 GUI and event stream
- MVP 3 Implementation Pipeline
- MVP 4 Brainstorming Room and Decision Record
- MVP 5 Memory / Decision Log
- MVP 6 Risk Radar / Quality Center

## Alpha 0.1 Starts From This Baseline

Alpha 0.1 focuses on:

- current README/getting-started/configuration/architecture docs
- dogfooding runbook and smoke
- LLM structured WorkPackage path with explicit fallback reason
- SQLite schema version and migration metadata
- Command Center summary
- quick/full verification matrix

## Known Limitations

- LLM PatchSet generation remains pending after this baseline note.
- LangGraph checkpointing remains pending.
- Hosted deployment, external CI ingestion, external scanners, vector DB by
  default, multi-user collaboration, and release packaging remain post-Alpha
  topics.
